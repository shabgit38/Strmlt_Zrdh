* Fetches holdings from Zerodha Kite

3. Rule-based alerts

# Job Logic

## fetch_holdings_job()

Input:

Kite holdings API

Output:

data/holdings_snapshot.csv

Store:

tradingsymbol
quantity
avg_price
invested_amount

Handle API failures safely.

---

## fetch_ltp_job()

Input:

symbols from holdings snapshot

Fetch:

latest prices only

Output:

data/ltp_snapshot.csv

Columns:

timestamp
tradingsymbol
LTP

Do not re-fetch holdings.

---

## generate_alerts_job()

Input:

holdings + LTP + indicator data

Generate alerts:

1.

Near EMA20

Rule:

abs(LTP−EMA20)/EMA20 <=1%

2.

Near 52W High

Rule:

(52WHigh−LTP)/52WHigh <=3%

3.

Below EMA200

Rule:

LTP < EMA200

4.

High concentration

Rule:

allocation >15%

Output:

alerts.csv

Columns:

symbol
alert_type
message
timestamp
